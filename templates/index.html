API 사용 방법

이 API를 통해 위치와 키워드에 맞는 맛집을 추천받을 수 있습니다.

사용 예시:

/recommend?location=Anam Station, Seoul, South Korea&keyword=kalguksu

위의 요청을 보내면 다음과 같은 결과를 받을 수 있습니다:

[
    {
        "가게 이름": "소문난 칼국수",
        "주소": "서울특별시 성북구 고려대로 89",
        "평점": "4.3 (150 리뷰)",
        "장점": ["면발이 쫄깃하고 맛있어요!", "국물이 깔끔해요."],
        "단점": ["웨이팅이 길어요.", "주차가 어려워요."]
    }
]

맛집 추천 시 장단점은 리뷰를 기반으로 추출됩니다.

추가로 궁금한 사항이 있으시면 GPT를 통해 질문해 주세요!
